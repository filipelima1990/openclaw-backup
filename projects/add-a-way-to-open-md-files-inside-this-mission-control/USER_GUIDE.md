## Troubleshooting

### Preview Button Not Showing?

**The eye icon only appears for previewable files:**
- ‚úÖ `.md`, `.markdown`, `.html`, `.htm` files
- ‚ùå Other file types (`.txt`, `.json`, `.py`, etc.)

For non-previewable files, use the **link icon** to open in the file manager.

### Preview Opens Blank?

Possible causes:
1. File path is invalid
2. File has been moved or deleted
3. Permission issue (file outside allowed directories)

**Solution:** Use the link icon to open in file manager and check if the file exists.

### Preview Looks Wrong?

The preview uses a simple markdown parser. For complex markdown with:
- Tables
- Images
- Advanced formatting

You may see basic rendering instead. This is expected - use a full markdown editor for complex files.

Open multiple markdown files in separate browser tabs to compare them.

Use `Ctrl+P` (or `Cmd+P` on Mac) to print the preview. The styling is optimized for paper.
